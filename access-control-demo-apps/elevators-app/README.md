# IBM Cloud App ID
Node sample application that demonstrates the use of App ID's access control feature.

## Requirements
* Node 6.0.0 or higher

## Running Locally

Run the following commands:
```bash
npm install
npm start
```
Use the link http://localhost:3000 to load the web application in browser.
